# 🚀 바이브코딩 포트폴리오 프로젝트

이 프로젝트는 **[코딩알려주는누나 - 바이브코딩]** 강의의 스터디 과제 제출용으로 제작된 개인 포트폴리오 웹사이트입니다.

## 🔗 관련 링크
- **강의 정보**: [바이브코딩으로 실서비스 개발하기](https://codingnoona.thinkific.com/courses/vibe-coding)
- **배포 URL**: [https://study-vibe-coding-portfolio.vercel.app/](https://study-vibe-coding-portfolio.vercel.app/)

## 🛠 사용 기술
- **Language**: HTML5, CSS3, JavaScript
- **Deployment**: [Vercel](https://vercel.com)
- **Design Inspiration**: 사이드바 내비게이션 및 분할 레이아웃 기반 모던 디자인

## 📋 주요 기능
- **반응형 레이아웃**: 데스크탑 고정 사이드바 및 모바일 최적화 UI
- **Split Screen Hero**: 라이트/다크 테마가 공존하는 독특한 메인 화면
- **섹션 전환**: JavaScript를 이용한 부드러운 단일 페이지 애플리케이션(SPA) 스타일 전환
- **커스텀 아바타**: 사진이 없을 때를 대비한 CSS 기반 한국인 여성 개발자 아바타 구현
